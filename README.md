# AI Call Assistant

## 🚀 Overview

An AI-powered voice assistant that automates customer interactions using NLP and speech technologies, eliminating the need for manual keypad navigation.

## ✨ Features

* Speech-to-text using Whisper
* NLP-based intent recognition
* Text-to-speech response system
* Real-time interaction handling

## 🛠️ Tech Stack

* Python
* NLTK, SpaCy
* Whisper
* pyttsx3


## 📌 Future Improvements

* Integration with call systems
* Multilingual support
* Enhanced conversational AI
